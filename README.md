# Action Pickers for SillyTavern
This extension adds three popup dialogs where you can use your mouse to influence the passage of time, simulate a phone call, or choose from countless actions that will move the conversation forward.

Probably one of the least necessary extensions for anyone.

# Added commands (useful for QRs)

```timeskip``` - Displays a time skip popup where you can manage the time flow using an interactive UI.

```call``` - Displays a call popup where you can initiate a phone call request.

```actionselect``` - Displays an action selection popup where you can choose a desired action using from interactive UI.

# Installation
1. Open SillyTavern.
2. Click the "Extensions" icon.
3. Click "Download Extensions & Assets" and insert following link.
```
https://github.com/japkozjad/st-action-pickers
```
5. Click "Download".
6. Refresh the page.

This extension is compatible with SillyTavern 1.13+.
